On peut utiliser la Raspberry Pi 3 soit comme un central device (client GATT) soit comme un peripheral device (serveur GATT). Dans le cas dun appareil priphrique (peripheral device), on pourra grer ladvertising de la manire suivante :

// Activer l'advertising :
$ sudo hciconfig hci0 leadv 0

// Dsactiver l'advertising :
$ sudo hciconfig hci0 noleadv
